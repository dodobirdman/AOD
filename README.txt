App'en kører på en Express server og kræver Node.js til at starte.
Ved første brug, skal man bruge Node Package Manager og taste npm install i terminalen.
Dette vil installer alle nødvendige dependencies
Derefter i terminalen (med Bash installeret hvis man er på Windows) skal man skrive "node app.js" imens man er i AOD folderen
Serveren vil starte og websiden kan tilgås ved http://localhost:3000/

Brug af API til elpris er igennem en gratis API fra elprisenligenu.dk
